# SpaceIT 🚀
**Ecosistema de gestión de espacios de coworking - ITM**

## 📂 Estructura del Proyecto
Este repositorio contiene el diseño arquitectónico y los artefactos de la Entrega 2:
- `/docs`: Diagramas de arquitectura y bitácoras de aprendizaje.
- `/src/backend`: Estructura de clases (Entidades, Servicios, Controladores).
- `/src/database`: Script de diseño del modelo relacional (SQL).
